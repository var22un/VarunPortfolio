Web Design & development

Creates a website's look and feel. It includes aspects like webpage layout, content production, and graphic design. Web designers use design tools to create the visual elements and layout of a website.

Determines how a website functions. It involves using a combination of programming languages, tools, and frameworks to bring a website or web application to life. Web developers use programming languages like HTML, CSS, and JavaScript to translate designs into a functional website

UI refers to the way people interact with information systems. It includes the pages, screens, buttons, and other visual elements that users interact with. UI also refers to the touchpoints that people use to engage with a digital product. UX refers to how a user feels when they interact with a product or service. It includes a person's perceptions of utility, ease of use, and efficiency. UX design is the process of creating products or services that provide meaningful experiences for users.
